# David Celi Lupera - Abogado en Guayaquil

Sitio web profesional para servicios legales en Guayaquil, Ecuador.

## 🚀 Tecnologías

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Font Awesome Icons

## 📁 Estructura
```
├── index.html          # Página principal
├── css/
│   └── styles.css     # Estilos personalizados
├── js/
│   └── main.js        # Interacciones y funcionalidad
└── assets/
    └── images/        # Imágenes del sitio
```

## 🎨 Características

✅ Diseño responsive (mobile-first)
✅ Navbar fijo con efecto scroll
✅ Hero section con overlay
✅ Sección de servicios
✅ Carousel de clientes
✅ Formulario de contacto
✅ WhatsApp flotante
✅ Animaciones suaves
✅ SEO optimizado

## 🔧 Instalación

1. Clona el repositorio
2. Abre `index.html` en tu navegador
3. ¡Listo!

No requiere instalación de dependencias ya que usa CDN.
